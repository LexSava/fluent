# Design system

- Use shadcn/ui components as the base
- Dark theme by default via next-themes
- Font: Geist Sans (already included in Next.js)
- Color scheme: neutral slate palette with accent indigo
- All animations via framer-motion
- Mobile-first, fully responsive layouts
- Chat interface should feel like a modern messenger app
- Prefer clean, minimal UI — no unnecessary decorations
- Use lucide-react for all icons
