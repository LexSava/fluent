# Workflow rules

## Working directory
Work only inside this project folder. Do not touch any files outside the project.

## Allowed without confirmation
- Reading and editing files inside the project
- Running npm/npx commands
- Running prisma migrate, prisma generate
- Running npm run dev / build / lint / test

## Requires my confirmation
- Deleting any files
- Installing new npm packages (ask me first)
- Any git commands (commit, push, reset, rebase)
- Any changes to .env files

## Forbidden
- Do not read or modify files outside the project folder
- Do not make requests to external URLs
- Do not modify system files
- Do not run git push unless explicitly asked
